# Markdown in 5min

## Headers

```plain
# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header
```

## Text formatting

```plain
**bold text**
*italic text*
***bold and italic***
~~strikethrough~~
`inline code`
```

## Links and images

```plain
[Link text](https://example.com)
[Link with title](https://example.com "Hover title")
![Alt text](image.jpg)
![Image with title](image.jpg "Image title")
```

## Lists

```plain
Unordered:
- Item 1
- Item 2
    - Nested item

Ordered:
1. First item
2. Second item
3. Third item
```

## Blockquotes

```plain
> This is a blockquote
> Multiple lines
>> Nested quote
```

## Code blocks

````plain
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## Tables

```plain
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```

## Horizontal rule

```plain
---
or
***
or
___
```

## Task lists

```plain
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
```

## Escaping characters

```plain
Use backslash to escape: \* _ \# \`
```

## Line breaks

```plain
End a line with two spaces
to create a line break.

Or use a blank line for a new paragraph.
```
